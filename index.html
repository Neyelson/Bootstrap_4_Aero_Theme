Bootstrap Aero

Essa é uma versão modificada do Bootstrap 4.4.1 que apresenta elementos customizados com a temática Aero.

Funciona da mesma forma que o Bootstrap original, você vai precisar apenas da pasta BootstrapAero presente nos arquivos deste projeto. O tema pode ser aplicado mudando a cor de fundo dos elementos, para isso há duas opções novas disponíveis: a "bg-aeroblack" e a "bg-aerowhite".

Primeiros passos:

Para que haja um aproveitamento eficiente desse tema, é recomendável que seja respeitado alguns conceitos visuais específicos.

As transparencias ficam visualmente muito agradéveis quando estão por cima de imagens como planos de fundo, principalmente imagens com o efeito "blur". Esta página, por exemplo, já possui um plano de funco aplicado em seu elemento "body". Há um pedaço de código abaixo que você pode utilizar para aplicar esse mesmo efeito facilmente.

body {
background-image: url ( https://i.imgur.com/VPEH3zV.jpg ) ;
background-color: grey ;
background-size: 100% 100% ;
background-attachment: fixed ;
background-position: center ; }
Botões:

Ao aplicar o "bg-aeroblack" ou o "bg-aerowhite" nos botões, eles se tornarão transparentes. Ao passar o ponteiro do mouse por cima, os botões vão mostrar as suas respectivas cores originais.

Cartões:

Ao aplicar o "bg-aeroblack" ou o "bg-aerowhite" nos cartões, eles se tornarão transparentes. Ao adicionar elementos Aero dentro de cartões, ou cartões dentro de cartões, tais elementos ficarão menos transparentes e mais realçados.
